Vocalizer Automotive 5.5 Driver

This add-on bridges the legacy 32-bit Nuance Vocalizer Automotive 5.5 driver to 64-bit NVDA 2026.1 and newer.

The original Vocalizer Automotive 5.5 project is abandonware. The original add-on author is not responsible for this fork, its modifications or its support.

Requirements

The package includes the runtime components required by the 32-bit Automotive host. It does not include separate voice add-ons or the user-specific license file.

  • vautov5.dll
  • nuan_platform.dll
  • nuan_platform.dllz and the required component data.
  • Required tuning files.
  • At least one compatible Vocalizer Automotive voice add-on.
  • A valid Vocalizer license for this computer.

Installation

  1. Install the add-on.
  2. The add-on already contains the required Automotive runtime components.
  3. Install your voice add-ons separately.
  4. Use NVDA menu > Vocalizer Automotive to enter the license and configure automatic language switching.
  5. Restart NVDA and select vocalizerAutomotive32.

The license file is not included in the package and must be imported from your own licensed copy.

Audio processing

This release uses the classic NVDA compatibility bridge. Sonic Pitch remains independent and uses its own WavePlayer hook.

Release variants

  • v2.1.7: classic bridge for NVDA 2026.1 and newer.
  • v2.2.0: brokered-audio bridge for NVDA 2026.2 and newer.

Install only one variant at a time.

Automatic language switching

The settings dialog detects installed voices from their .hdr metadata and stores the selected mapping in %APPDATA%\nvda\vocalizer.ini.

Translations

The add-on interface includes the original translations for locales an, ar, da, de, el, es, fi, fr, gl, hr, hu, it, ja, ko, nb_NO, ne, nl, pl, pt_BR, pt_PT, ru, sk, sl, tr and zh_CN. HTML documentation is available in English, Polish and Slovak.
